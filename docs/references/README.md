# 论文分类与报告交叉引用索引

本目录仅归档本作品直接使用的论文副本。原始资料目录保持只读，不移动、不修改。

## 分类

- `multi_agent/`：多 Agent 失败机理、通信攻击、级联错误、恶意节点、时序图检测、拜占庭容错与事务回滚。
- `RAG/`：RAG 基础、知识投毒攻击、防御与溯源、事实验证、图谱增强和可信 RAG 综述。

## 报告使用原则

1. 背景与相关工作引用论文原始问题定义和公开实验结论。
2. 本文算法只声明“受某论文启发”或“在其基础上组合”，不把已有方法写成原创。
3. 报告中的本地实验结果来自 `zyjd_system/experiments/results/`，不冒充论文复现实验或公开数据集成绩。
4. 论文题名、作者、年份和链接以 PDF 首页、arXiv、ACL Anthology 或正式会议页面为准。
