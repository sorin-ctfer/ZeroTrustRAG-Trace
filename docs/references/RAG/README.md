# RAG 论文归档

## 基础检索与生成

- RAG (Lewis et al., NeurIPS 2020)：检索增强生成基本框架。

## 知识投毒攻击

- PoisonedRAG：检索条件与生成条件共同满足的知识投毒机理。
- Pandora、Phantom、Poisoned-MRAG、UniC-RAG：不同投毒载体、触发方式和多模态扩展。

## 防御、溯源与鲁棒性

- RAGForensics / Traceback：从错误输出回溯候选污染文本。
- RobustRAG、RAGuard、FilterRAG、Towards More Robust RAG：鲁棒聚合、过滤和安全框架。
- Benchmarking Poisoning Attacks：攻击、防御与评测设置比较。

## 事实验证与可信评价

- RAGAS、ARES、FActScore：忠实度、事实性和自动评价。
- Trustworthiness in RAG、Towards Trustworthy RAG、Towards Secure RAG：可信与安全 RAG 综述。

## 图谱增强

- GraphRAG、GraphRAG Survey、HyperGraphRAG：图结构建模。本文只借鉴图表示方法，用于安全溯源而非普通问答优化。
